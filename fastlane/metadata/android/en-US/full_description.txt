<i>freeDictionaryApp</i> is an android dictionary for <a href='https://dictionaryapi.dev/'>Free Dictionary API</a> and <a href="https://wiktionary.org/">Wiktionary API<a/>. You can just use the search input to search the word.

<b>Features</b>
<ol>
    <li>English to English dictionary</li>
    <li>Supports multiple languages to English using Wiktionary API (case-sensitive)</li>
    <li>Definition of the word</li>
    <li>Example of the word usage if available</li>
    <li>Synonyms/Antonyms of the work if available</li>
    <li>Pronunciation of the word, both IPA text and audio using TTS</li>
    <li>Save searched data for offline uses</li>
    <li>Search for a random word</li>
</ol>
